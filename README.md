### Installation
1. Clone the repo
   ```sh
   git clone https://github.com/a-bemporad/whatslab.git
   ```
2. Install dependencies
   ```sh
   npm install
   ```
   or 
    ```sh
   yarn install
   ```
3. Start the project
   ```sh
   npm start
   ```
   or 
    ```sh
   yarn start
   ```
